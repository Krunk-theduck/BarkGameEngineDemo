# Directory Structure

## Root
- **index.html**: Main entry point and canvas container
- **main.js**: Game initialization and setup

## Engine Core (/engine)
- **engine.js**: Core engine functionality, update loops, entity management
- **scene.js**: Scene management and map-related functionality
- **entities.js**: Entity classes including Player controller
- **camera.js**: Camera and viewport management

## Game Scripts (/scripts)
- **entities/**: Custom entity scripts
- **behaviors/**: Reusable behavior components
- **utils/**: Helper functions and utilities

## Assets (/assets)
- **sprites/**: Game sprite images
- **audio/**: Sound effects and music
- **maps/**: Map data and tilesets

## Notes (/.notes)
- **project_overview.md**: High-level project description
- **meeting_notes.md**: Development progress and decisions
- **task_list.md**: Project tasks and progress
- **directory_structure.md**: This file
